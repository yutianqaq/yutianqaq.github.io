网络安全入门第一站——Hackthebox 演练平台

#Hackthebox
Dec 3, 2023

介绍

HackTheBox 提供了一个虚拟的靶场环境，可以亲自动手进行网络攻防实践。通过解决各种题目和挑战，可以模拟真实的攻击场景，学习和应用各种网络安全技术和工具。帮助加深对网络安全概念和技能的理解和掌握。

HackTheBox 提供了各种不同难度级别的挑战，涵盖了多个领域，如网络渗透测试、逆向工程、域环境渗透等。这些挑战涉及各种技术和技能，可以全面地学习和实践不同方面的网络安全知识。

初学者

对于初学者，建议在 Hack The Box :: Starting Point 完成关于基础渗透测试方法、工具练习。只需要有一些计算机基础知识就可以开始。

起点（Starting Point）中的 Archetype 是起点的第一台靶机。在起点中的靶机会以 标签（Tags）的方式来标记本章将学习/讲解的知识点。还提供了完整的攻略来帮助你完成靶机。

连接靶机有 Pwnbox、OpenVPN 两种方式连接。这里建议使用 OpenVPN 方式，因为图形化的 Pwnbox 延迟较高，操作不流畅。

入门

当完成 Hack The Box :: Starting Point 中的靶机后就可以做一些 Active/Retired 中的容易难度靶机。

Active 靶机分类

其中在 Active 靶机分类中是一些赛季靶机，作为一种挑战存在。没有完整攻略，需要靠自己的力量来完成挑战。

其中 MACHINE 表头是靶机的名字和靶机操作系统类型。
DIFFICULTY RATING 表头标识社区认为的难度。绿色为简单，黄色为中等，红色为困难。
RATING 表头表示社区评分，分高的接近实战，分低的是 CTF 风格。

Retired 靶机分类

在 Retired 靶机分类中是赛季/Active 靶机退役后的分类，仅限 VIP/VIP+ 使用。

不过，在退役靶机分类中会有一些带有 FREE 标签的免费机器。免费机器是轮换的。

以 Broker 机器为例，退役靶机分为 Adventure 模式和 Guided 模式。Guided 模式为指引模式。会以提问的形式一步一步指引你获取 USER FLAG，ROOT FLAG。

网络延迟

由于靶机服务器在国外所以国内裸连延迟较高，会影响扫描、爆破速度。
所以这里一般选择新加坡（SG）节点 。

在挂载前置节点后，延迟可以降到 110、120 左右。受地理位置影响，靠近新加坡的位置可能延迟更低。
