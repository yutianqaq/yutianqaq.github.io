BOF 是什么

Beacon Object Files (BOFs) 于 2020 年在 Cobalt Strike 4.1 中引入。自发布以来，BOF 在后利用活动中发挥了关键作用，超越了反射 DLL、.NET 程序集和 PowerShell 脚本。
