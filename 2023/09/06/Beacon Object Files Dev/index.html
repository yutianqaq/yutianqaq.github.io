概述

今天读了 Simplifying BOF Development: Debug, Test, and Save Your Bacon | Cobalt Strike 是 Cobalt Strike 官方发布的一篇简化 BOF 开发。

BOF 是什么

Beacon Object Files (BOFs) 于 2020 年在 Cobalt Strike 4.1 中引入。自发布以来，BOF 在后利用活动中发挥了关键作用，超越了反射 DLL、.NET 程序集和 PowerShell 脚本。
