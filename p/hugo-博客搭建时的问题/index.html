Nov 25, 2024
1 minute read

不会自动换行

打开 hugo 博客的文件夹，全局搜索 goldmark
如果是 toml 格式，就改为下面的内容

1
2
3

[goldmark.renderer]
unsafe = true
hardwraps = true

yaml 格式

1
2
3
4

markup:
  goldmark:
    renderer:
      hardWraps: true
