# 系统和软件包自动换源
```
curl -LO https://gitee.com/RubyMetric/chsrc/releases/download/pre/chsrc_latest-1_amd64.deb
sudo sudo dpkg -i chsrc_latest-1_amd64.deb
```


# docker
```
curl https://get.docker.com/ | sh

# 换源（可选）
sudo chsrc set dockerhub
```
